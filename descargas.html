HOJA DE PERSONAJE

Hoja de referencia para los trasfondos de cada personaje jugador.

Cada escenario te va a proporcionar las opciones para que puedas crear tu personaje según el trasfondo que hayas elegido. En esta sección vas a poder descargar la Hoja de Personaje necesaria para poder anotar las opciones de tu trasfondo y otras cosas que surjan durante el juego. El archivo viene en formato PDF por lo cual vas a necesitar un lector de archivos que pueda abrir el formato.

HOJAS DE ESCENARIOS

Hojas necesarias para crear escenarios y sus comunidades para el juego.

En Leyenda podés crear cuatro tipos de escenarios distintos. Acá te proveemos las hojas para cada uno de ellos y también para crear la comunidad central de cada juego. Cada archivo se encuentra en formato PDF y todos vienen compilados en formato RAR, vas a necesitar aplicaciones específicas para poder abrir ambos formatos.

RESUMEN DE REGLAS

Acá tenés todas las reglas que puedas llegar a necesitar durante una partida.

En este PDF compilamos todas las reglas importantes de Leyenda, junto a otras referencias, para que puedas tener fácil acceso en tus partidas. Están todas en forma de tarjeta para poder tener a mano durante el juego e incluyen las páginas donde podés encontrar dicha regla de ser necesario consultar el libro.

GUÍA DE INICIO RÁPIDO

Si querés jugar Leyenda y no tenés el libro esta guía te va a ayudar a empezar.

Esta es la Guía de Inicio Rápido para Leyenda, en ella vas a encontrar todo lo que necesitás para poder tener tu primer experiencia con el sistema de juego. Sus reglas básicas y un escenario breve te van a permitir jugar tu primer partida de Leyenda antes que puedas tener el libro. Queremos que conozcas el juego y te emociones tanto como nosotros.

STICKERS DADO LEGENDARIO

Stickers con las distintas caras del Dado Legendario para que puedas imprimir.

Acá podés encontrar el Dado Legendario en forma de stickers, de esta manera siempre vas a poder tener acceso a uno sin la necesidad de tenerlo en físico. Con este archivo vas a poder imprimir las caras del Dado Legendario y pegarlas a cualquier dado de doce caras. El formato del archivo es AI (Adobe Illustrator), que es el que suelen pedir las imprentas para stickers.
